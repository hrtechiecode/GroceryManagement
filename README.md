# Grocery Management System

## Tech Stack
- Java (Spring Boot)
- React.js
- MySQL

## Features
- User Login & Signup
- Admin Panel
- Add Products
- Order Management

## How to Run
Backend:
mvn spring-boot:run

Frontend:
cd grocery-frontend
npm install
npm start
